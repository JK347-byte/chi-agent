# FB 貼文五種文體模板

每種文體結尾固定格式：
```
本文資訊僅供衛教參考，不取代專業醫療診斷與治療建議。如有症狀請諮詢醫師。

🔖 來源引用：[論文標題或指引名稱，含年份]
#主題tag #日和聯合診所 #詹士奇醫師

--- 內部備註（給醫師審閱用）---
⚠️ 建議醫師確認：[需確認的醫學細節]
💡 延伸發展：[後續可延伸的貼文方向]
```

## A｜迷思破解型（分享率最高）

適用：「你以為 X，其實是 Y」的反直覺題材

```
【[點出迷思的吸睛標題]】

▋ [診間觀察或問題背景]
[2–3 句描述常見誤解，引發共鳴]

▋ [破解迷思的段落標題]
[正確觀念，1–2 句說清楚]

▋ 為什麼會這樣？
[醫學解釋，2–3 段，深入淺出]

▋ [實用建議段落標題]
● [建議一]（無順序用 ●）
● [建議二]
● [建議三]

或

1️⃣ [步驟一]（有順序用 1️⃣2️⃣3️⃣）
2️⃣ [步驟二]
3️⃣ [步驟三]

▋ 詹醫師的溫馨提醒
[1–2 句親切收尾]

如果你也有這個疑問，歡迎留言或來診所聊聊 😊

讓我們把關你的健康。
```
